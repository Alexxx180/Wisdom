# Венец

Программа работает только с помеченными редактором записями.

## Выделение записи

Нажмите по не выделенной записи левой кнопкой мыши.

## Снятие выделения

Нажмите по выделенной записи левой кнопкой мыши.

## Снять пометки

1. Выделите записи.
2. Нажмите «+».

## Удалить помеченные записи

1. Выделите записи.
2. Нажмите «X».
